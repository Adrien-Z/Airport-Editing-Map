About

This interactive map shows all airports which have been redone by the Infinite Flight Airport Editing team. This means that all airports you see on this map are going to be in high-quality in Infinite Flight. Any issues regarding a reworked airport? Feel free to report the issue by using our dedicated form.

If you are interested in contributing to the project, please visit airportediting.com.
