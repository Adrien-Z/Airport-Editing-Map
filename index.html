Additional information

This interactive map shows all airports which has been redone by the Airport Editing team. This means that all airports you see on this map are going to be in high-quality in Infinite Flight. If it’s not here, no worries, all airports are still included in global flight. They just won’t look as realistic.

If you want to help contribute to the project, please visit airportediting.com.
